# Code / Workflow Comparisons

This directory is for public examples comparing a naive one-off agent prompt/script with the skill-guided workflow.

Suggested comparison format:

1. **Task** — what the user wanted.
2. **Naive approach** — single prompt or script, likely failure modes.
3. **Skill-guided approach** — loaded skill, context gathering, quality gates, verification.
4. **Outcome difference** — concrete artifact quality, fewer repeated instructions, better stopping rule.
5. **Reusable lesson** — what the skill encodes.

Do not include private project data unless explicitly redacted or replaced with synthetic examples.
